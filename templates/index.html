StartUp-Compass Flask API Documentation

GET /get_google_trends

Fetch Google Trends data, interpret it, and return the requested section (e.g., trend summary, spike events, breakpoints, recommendations).

Query Parameters:

Parameter Type Description Default
niche String The search term for Google Trends. iced coffee
location String The location (e.g., US, PH). US
timeframe String Time range for trends (e.g., today 12-m for 1 year). today 12-m
request_type String Type of data requested (e.g., trend_summary, spike_events, breakpoints, recommendations). trend_summary

Example Response:

{
    "Upward": 30,
    "Downward": 12
}

Parameter Type Description Default

timeframe String Time range for trends (e.g., today 12-m). today 12-m

Example Response:

{
    "trend_summary": { "Upward": 30, "Downward": 12 },
    "spike_events": { "2024-09-01": { "Interest Score": 95 } },
    "breakpoints": [{ "breakpoint_index": 100 }],
    "recommendations": ["Consider expanding product offerings."]
}

GET /get-bar-graph-data

Retrieve data for creating a bar graph showing interest scores over time for a specific niche based on Google Trends data.

Query Parameters:

Parameter Type Description Default
niche String The search term for Google Trends. iced coffee
location String The location for the trends data (e.g., US, PH). US
timeframe String Time range for trends data (e.g., today 12-m). today 12-m

Example Response:

The response will return the data in JSON format, suitable for creating a bar graph. The labels represent the dates (X-axis), and the values represent the interest scores (Y-axis).

{
    "labels": [
        "2023-09-01",
        "2023-09-02",
        "2023-09-03",
        "... (more dates)"
    ],
    "values": [
        67, 
        73, 
        82, 
        "... (more scores)"
    ]
}

Error Response:

If the API cannot find data for the given niche, it will return an error message:

{
    "error": "No data found for the given niche."
}

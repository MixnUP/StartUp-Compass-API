StartUp-Compass Flask API Documentation

GET /get_google_trends

Fetch Google Trends data, interpret it, and return the requested section (e.g., trend summary, spike events, breakpoints, recommendations).

Query Parameters:

Parameter Type Description Default
niche String The search term for Google Trends. iced coffee
location String The location (e.g., US, PH). US
timeframe String Time range for trends (e.g., today 12-m for 1 year). today 12-m
request_type String Type of data requested (e.g., trend_summary, spike_events, breakpoints, recommendations). trend_summary

Example Response:

{
    "Upward": 30,
    "Downward": 12
}

GET /calculate_roi

Calculate the estimated Return on Investment (ROI) based on Google Trends forecast for a specific niche.

Query Parameters:

Parameter Type Description Default
niche String The search term for Google Trends. iced coffee
location String The location (e.g., US, PH). US
timeframe String Time range for trends (e.g., today 12-m). today 12-m
investment_amount Float The amount of money being invested. 1000
forecast_period Integer Number of days into the future to forecast. 90

Example Response:

{
        "niche": "iced coffee",
        "investment_amount": "$1000",
        "forecast_period_days": "90 days",
        "initial_interest_score": 45.7,
        "forecasted_interest_score": 50.2,
        "estimated_roi": "1100.45%",
        "interest_growth": "9.81%",
        "growth_trend": "upward",
        "recommendation": "Based on the trend, the interest in 'iced coffee' is expected to grow. Your estimated ROI for a $1000 investment over 90 days is 1100.45%. NOTE: This calculation is based on the percentage growth of the interest score over a period of time."
    }

GET /get-bar-graph-data

Retrieve data for creating a bar graph showing interest scores over time for a specific niche based on Google Trends data.

Query Parameters:

Parameter Type Description Default
niche String The search term for Google Trends. iced coffee
location String The location for the trends data (e.g., US, PH). US
timeframe String Time range for trends data (e.g., today 12-m). today 12-m

Example Response:

The response will return the data in JSON format, suitable for creating a bar graph. The labels represent the dates (X-axis), and the values represent the interest scores (Y-axis).

{
        "labels": [
            "2023-09-01",
            "2023-09-02",
            "2023-09-03",
            "... (more dates)"
        ],
        "values": [
            67, 
            73, 
            82, 
            "... (more scores)"
        ]
    }

Error Response:

If the API cannot find data for the given niche, it will return an error message:

{
        "error": "No data found for the given niche."
    }

GET /business-assessment

Submit business metrics as query parameters to receive tailored insights and recommendations based on your current financial performance, including growth rate, profit margin, and industry benchmarks.

Query Parameters:

Parameter Type Description Required
current_revenue Number The total revenue for the current period (e.g., year, quarter). Yes
previous_revenue Number The total revenue for the previous period, used for growth rate calculation. Yes
total_expenses Number The total expenses incurred during the current period. Yes
customer_base Number The total number of customers for the current period. Yes
months Number The number of months over which to calculate average revenue (default is 12). No
industry String The industry for which to fetch benchmarks (e.g., 'Retail', 'Tech'). Yes

Example Request:

Below is an example of a request URL with query parameters:

    GET /business-assessment?current_revenue=120000&previous_revenue=100000&total_expenses=80000&customer_base=300&months=12&industry=Retail

Example Response:

The response will return insights along with financial metrics in JSON format:

{
        "insights": [
            "A low growth rate indicates potential stagnation. Explore new market segments or diversify your product offerings.",
            "The average revenue per month is $10000.00.",
            "Your revenue is relatively low. Consider implementing aggressive marketing strategies to attract new customers."
        ],
        "growth_rate": 20.0,
        "profit_margin": 33.33,
        "average_revenue_per_month": 10000.00,
        "message": "Business assessment completed successfully."
    }

Error Response:

If the request is missing required parameters or if any values are invalid, the API will return an error message:

{
        "error": "Missing required parameters."
    }

GET /trend_seeker

Retrieve the top keyword suggestions based on a specified keyword along with their current month interest data.

Notice: Please be aware that loading may take some time, especially for popular keywords.

Query Parameters:

Parameter Type Description Required
niche String The search term for which to fetch keyword suggestions. Yes
timeframe String The time range for trends data (default is 'today 12-m'). No
location String The geographical area for the trends data (default is 'US'). No
top_n Integer The number of top suggestions to return (default is 5). No

Example Request:

Below is an example of a request URL with query parameters:

    GET /trend_seeker?niche=coffee&timeframe=today%2012-m&location=US&top_n=5

Example Response:

The response will return the top suggestions along with their interest data in JSON format:

{
        "original_keyword": "coffee",

                    "latest_interest_score": 82,
                    "average_interest_score": 70,
                    "month": 9,
                    "year": 2024

            }
            // ... additional suggestions ...
        ]

Error Response:

If the request is missing required parameters or if any values are invalid, the API will return an error message:

{
